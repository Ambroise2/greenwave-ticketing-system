
# GreenWave Ticketing System

## Overview
A Python/Tkinter ticketing and session management system for the GreenWave Conference assignment.

## Run
1. Ensure Python 3.10+ is installed.
2. Optional: create and activate a virtual environment.
3. From the project root run:
```
python main.py
```

## Tests
Run:
```
python -m unittest discover -v
```

## Files
- `main.py` - entry point
- `src/` - source modules (models.py, storage.py, logic.py, gui.py)
- `tests/` - unit tests
- `data/` - pickle persistence files (auto-created)
